# kbs-cloud / pixelart

Welcome to the **kbs-cloud Pixel Art** repository. This repository serves as a centralized, categorized, and easily navigable storage for pixel art assets, sprite sheets, and tilemaps.

---

## 📂 Repository Index

To explore the assets, navigate through the folders below:

* 🎨 **[sprites/](sprites/README.md)**: Categorized and clean sprite sheets organized by theme/type.
  * 🧪 [potions-gems-jewels/](sprites/potions-gems-jewels/README.md): Sprites for potions, flasks, gems, and jewelry.
  * ⚔️ [action-bar/](sprites/action-bar/README.md): System menu icons, action bar buttons, and UI interface sprites.
* 📥 **[downloads/](downloads/README.md)**: Contains the original raw downloaded files with their raw timestamps from source generation.
