# Categorized Sprite Sheets & Assets

This section of the repository hosts clean, categorized pixel art sprite sheets. The folders are organized by theme and functional use-case.

---

## 📂 Sprite Categories

### 🧪 [Potions, Gems, & Jewels](potions-gems-jewels/README.md)
* **Description**: RPG inventory and item assets. Contains potions of various shapes/colors, magic crystals, diamonds, and valuable jewelry (rings, pendants).
* **Format**: Spritesheet (grid layout).
* **Go to Folder**: [`potions-gems-jewels/`](potions-gems-jewels/README.md)

### ⚔️ [Action Bar & System Icons](action-bar/README.md)
* **Description**: Game UI icons. Contains buttons and menu indicators for settings, quest lists, inventory screens, skills, status effects, and generic combat/hotbar interface actions.
* **Format**: Multi-part sprite sheets.
* **Go to Folder**: [`action-bar/`](action-bar/README.md)

---

## 🛠️ Usage Guidelines
* If importing into game engines (like Unity, Godot, or Unreal), configure the texture import settings to **Sprite (2D and UI)** or use point-filtering (no compression/filtering) to preserve the crisp retro pixel look.
* Slice the grids using cell-based cropping or automatic pixel grid alignment.
